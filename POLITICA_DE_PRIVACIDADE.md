# Política de Privacidade

**Última atualização:** Maio de 2026

Bem-vindo ao bot **TribalWars Maps** para Discord. Ao utilizar este bot, você concorda com a coleta e o uso de informações descritas nesta Política de Privacidade.

## 1. Coleta de Informações
Nosso bot acessa informações básicas disponibilizadas pela API do Discord estritamente necessárias para a execução de seus comandos, incluindo:
- **IDs de Servidores (Guild IDs)** e **IDs de Canais (Channel IDs):** Para armazenar em qual servidor e canal as atualizações diárias (mapas) devem ser enviadas.
- **Interações de Comandos:** Para responder adequadamente aos comandos de barra (`/`) e menções acionados pelos usuários.

**O que NÃO coletamos:**
- Não armazenamos mensagens de texto de nenhum canal.
- Não coletamos nomes de usuário, endereços de e-mail, senhas, ou qualquer dado pessoal sensível.
- Não rastreamos o tempo online ou os status dos usuários no servidor.

## 2. Uso das Informações
As informações coletadas (IDs de servidores e canais) são armazenadas localmente no banco de dados do bot exclusivamente para:
- Viabilizar a funcionalidade de relatórios diários de mapas do Tribal Wars nos canais configurados.
- Permitir que os administradores do servidor personalizem quais mapas desejam receber.

## 3. Retenção de Dados
Os dados de configuração de canal são mantidos apenas pelo período em que o bot estiver operante naquele servidor. Caso o administrador deseje interromper o serviço, um novo comando ou a remoção do bot fará com que o bot deixe de publicar naquele canal. Os dados órfãos podem ser removidos periodicamente do banco de dados do bot.

## 4. Compartilhamento de Dados
Nenhuma informação do seu servidor ou configurações do bot é vendida, alugada ou compartilhada com terceiros. Os dados existem unicamente na infraestrutura onde o bot é hospedado.

## 5. Contato
Para remover as informações do seu servidor do nosso banco de dados, você pode expulsar o bot do seu servidor. Se houver dúvidas referentes a esta Política de Privacidade, entre em contato com o desenvolvedor responsável.
